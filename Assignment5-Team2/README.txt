# CSCI-P532
This repository holds the work of Peter, and Ramya, and Aditi for Assignment 4 -- Creating a Game Maker

How to run this application:
	1. Install JavaFx and add it to your system path.
	2. Install Java 11
	3. Change directory to Team2-Week4\src\main\java\gamemaker\controllers\ParentWindow.java
	4. Run the following command: `javac --module-path %PATH_TO_FX% --add-modules javafx.controls ParentWindow.java`
